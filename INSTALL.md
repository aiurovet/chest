HOW TO INSTALL AND UNINSTALL

Linux (via Linuxbrew) and macOS (via Homebrew):

- First time: `brew tap aiurovet/chest && brew install chest`
- Otherwise:  `brew update chest`
- Uninstall:  `brew uninstall chest && brew untap aiurovet/chest`

Windows (via Chocolatey, use `-y` for auto-confirmation):

- First time: `choco install -y chest`
- Otherwise:  `choco update chest`
- Uninstall:  `choco uninstall chest`
